# 内置类型

## Python3 中内置类型

- bool
- int
- float
- complex
- str
- list, tuple
- dict
- set, fronzenset

## 布尔值 bool

> None、0、空字符串、以及没有元素的容器都为 False, 反之为True

## 数字 int

## 字符串 str
> 字符串是不可变类型

## 列表 list

`一种可变的容器类型`

> 在Python的官方实现中，list是一种采用分离式技术实现的动态顺序表


## 元组 tuple

> 元组是跟列表非常接近的容器类型
> 但与列表一个很重要的区别是元组是一种不可变类型
